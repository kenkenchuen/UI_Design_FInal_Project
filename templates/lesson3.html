Lesson 3 - Character Classes

Character classes are indicated by square brackets, and are used to match any character inside the brackets. For example, “[abc]” will match either a, b, or c.
Using a hyphen inside a character class specifies the characters' range. For example, “[a-d]” is the same as “[abcd]”. Note, the order of the characters or the range doesn't matter.

Pattern
Text
"fo[z-a]"
⇒
They were cannon fodder before the four battleships.
“[0-9]”
⇒
There are only 28 days in February.
